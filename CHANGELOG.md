# Changelog

## 0.4.14 - 2026-06-16

### 新功能

- 管理员上线准备度页新增 YouTube 数据管道观测，展示 `GOOGLE_API_KEY`、语义 QA、Raw 抓取、QAAuditLog 覆盖、ActivityEvent 入库和 YouTube 卡片生成状态。
- 内容维护页在选择 YouTube 来源时提示当前按钮走标准人物刷新链路，批量 Raw 抓取、QA 清洗和 ActivityEvent 入库状态统一到上线准备度页查看。

## 0.4.13 - 2026-06-16

### 修复

- 修复增量内容抓取脚本的运行路径、环境加载和 YouTube API 前置检查，支持按来源、人物和数量重跑，并记录 YouTube 失败原因。
- 新增本轮 YouTube 后处理脚本，把最新抓取内容补齐 QA 审计、processed 标记和动态物化，只将最新 QA verdict 为 keep 的视频推入 `ActivityEvent`。
- 修复 YouTube 描述含 emoji 时摘要截断切断 surrogate pair，导致 Prisma 写入动态时报 `unexpected end of hex escape` 的问题。

### 性能优化

- 优化首页、话题页、机构页、关系图和对比报告相关查询与组件加载，降低首屏和实体页的数据读取压力。

## 0.4.12 - 2026-06-15

### 新功能

- 新增个人空间和后台管理的侧栏式工作区布局，关注、对比报告、邮件订阅和管理员页面统一在同一套导航里切换。
- 对比报告详情页纳入个人空间布局，并把报告链路改成“可信度链路”说明，减少工具过程对读者的干扰。
- 对比报告证据区拆成“观点引用证据”和“补充来源”，矩阵里的证据入口直接展示人物和来源标题。

### 修复

- 关注页移除重复主站头部，由个人空间布局承载导航，避免账号入口和工作区入口互相打架。
- 账号安全页纳入个人空间布局，但不再出现在账号菜单和个人空间常用导航里。

## 0.4.11 - 2026-06-15

### 修复

- 补齐生产数据库的内容维护迁移，修复进入“内容维护”时 `MaintenanceSchedule` 表和维护任务字段缺失导致的页面报错。
- 管理员菜单去掉“后台首页”，按内容维护、账号管理和上线运维相关性重新排序，并预取菜单目标页。
- 账号菜单复用短期登录态缓存，切换管理员页面时不再闪回未登录外观。
- 内容维护首屏只查询页面实际使用字段，并压缩任务、规则和日志列表的初始加载量。

## 0.4.4 - 2026-06-14

### 修复

- 修复本地开发环境下 Next 内部 HMR 路径被 auth proxy 截获，导致登录页无法完成 hydration、按钮点击无响应的问题。
- 为 Arco feedback API 注入 React 19 `createRoot`，避免注册、登录和忘记密码提示触发 `ReactDOM.render` 兼容错误。
- 修正 NextAuth credentials 失败结果被误判为登录成功的问题，错误账号密码现在会展示登录失败提示。
- 补充 auth 页面的回归断言，覆盖注册提示 hydration、Arco bridge 和登录错误判断顺序。

## 0.3.0 - 2026-06-14

### 新功能

- 顶部内容导航新增独立的“人物对比”入口，保留“人物关系”并让对比入口紧跟其后。
- 关注页把人物、话题和机构统一成“关注对象”视图，并优化本地关注与账号关注的合并逻辑。
- 首页推荐动态和搜索入口改成更轻的同区块布局，减少首屏信息挤压。

### 性能优化

- 人物详情页改为缓存核心数据，并对关系图、近期动态、视频和课程区块做可见后加载。
- 人物内容与课程 API 限制返回字段和分页边界，降低详情页附属请求负载。
- Neon WebSocket 依赖改为运行时 require，并默认关闭 bufferutil，减少生产冷启动兼容风险。

### 修复

- 修复“我的对比”和公开对比报告在导航高亮上的语义混用。
- 修正响应式 smoke 对人物关系页旧文案的检查，跟当前页面标题保持一致。
