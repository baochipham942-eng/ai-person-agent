# Changelog

## 0.4.8 - 2026-06-14

### 修复

- 修复邮箱验证暂停后，旧版已创建但仍处于待验证状态的账号重新注册时仍被“该邮箱已注册”挡住的问题；这类账号现在会在重试注册时自动激活并继续登录。

## 0.4.7 - 2026-06-14

### 修复

- 暂停新用户注册的邮箱验证拦截，注册成功后账号直接激活并自动登录，方便邀请用户快速进入产品。
- 修正认证邮件链接的生产域名优先级，并把登录、注册、重置密码和邮件 CTA 的主按钮统一为黑色。

## 0.4.6 - 2026-06-14

### 修复

- 允许 `people.llmxy.xyz` 反代域名调用 Next Server Actions，修复正式域名注册提交被 CSRF origin 校验拦截并显示 Server Components 泛化错误的问题。

## 0.4.5 - 2026-06-14

### 修复

- 修复生产 standalone 环境下注册事务超时导致注册失败的问题，密码 hash 会在开启数据库事务前完成，并收敛注册失败时暴露给用户的错误信息。

## 0.4.4 - 2026-06-14

### 修复

- 修复本地开发环境下 Next 内部 HMR 路径被 auth proxy 截获，导致登录页无法完成 hydration、按钮点击无响应的问题。
- 为 Arco feedback API 注入 React 19 `createRoot`，避免注册、登录和忘记密码提示触发 `ReactDOM.render` 兼容错误。
- 修正 NextAuth credentials 失败结果被误判为登录成功的问题，错误账号密码现在会展示登录失败提示。
- 补充 auth 页面的回归断言，覆盖注册提示 hydration、Arco bridge 和登录错误判断顺序。

## 0.4.3 - 2026-06-14

### 修复

- 未登录头部隐藏“我的关注”和“我的对比”，并把个人入口收口到登录后的账号菜单。
- 首页本周推荐、搜索、筛选和排序重新拆分，保留“综合影响力”和“最近热度”两个可信排序入口。
- 注册页缺少邀请码时显示表单级错误并阻止提交。
- 人物详情页改为服务端解析 `section` 和 `highlight` 参数，移除客户端 `useSearchParams` 依赖。
- 对比报告入口回到新建报告页，并补齐 Agent 工具链展示和 readiness 权限测试。

## 0.4.2 - 2026-06-14

### 修复

- 未登录时右上角账号入口直接显示“登录”，不再先渲染空头像占位或显示“登录/注册”。
- 限制 Next 页面扩展名为 TypeScript，避免阿里云 FC 的 `proxy/index.js` 被误判成 `/proxy` 页面并触发 duplicate page 警告。

## 0.4.1 - 2026-06-14

### 修复

- 首页本周推荐改为左右箭头切换，移除点状分页和卡片隐式切换。
- 首页搜索入口移到“我的对比”左侧，并修复账号菜单弹层被横向滚动容器裁切的问题。
- 筛选和排序灰底改为跟随标签内容收缩，减少首屏筛选区的灰色占位。
- 登录页移除默认“重发验证”入口，验证邮件重发只保留在注册后的验证提示里。

## 0.4.0 - 2026-06-14

### 新功能

- 新增邮箱注册登录、邮箱验证、密码重置、一键登录设备管理和账号安全页。
- 新增管理员后台入口，覆盖用户管理、邀请码管理、审计日志、内容维护、质量复核、影响力校准、Newsletter 投递和上线准备度。
- 新增内容维护菜单，支持新人物构建、单人物/多人物/全站刷新、dry-run、execute、媒体渠道筛选、定时规则、任务详情、重试和取消。
- 新增右上角账号菜单，区分个人入口和管理员入口，并把账号安全、邮件订阅、后台管理和退出登录统一收口。
- 对比报告页和生成流程增强登录态处理、进度事件展示和来源覆盖说明。

### 修复

- 修复内容维护中 OpenAlex 单渠道更新缺少 ORCID 导致实际不抓论文的问题。
- 修正人物构建完成度按固定 6 个数据源计算的问题，改为按本次实际启用来源计算并限制在 100%。
- 后台管理 API 统一接入管理员权限边界和审计记录。

### 文档

- 同步账号、邀请码、审计、维护任务和定时维护相关数据库模型说明。

## 0.3.0 - 2026-06-14

### 新功能

- 顶部内容导航新增独立的“人物对比”入口，保留“人物关系”并让对比入口紧跟其后。
- 关注页把人物、话题和机构统一成“关注对象”视图，并优化本地关注与账号关注的合并逻辑。
- 首页推荐动态和搜索入口改成更轻的同区块布局，减少首屏信息挤压。

### 性能优化

- 人物详情页改为缓存核心数据，并对关系图、近期动态、视频和课程区块做可见后加载。
- 人物内容与课程 API 限制返回字段和分页边界，降低详情页附属请求负载。
- Neon WebSocket 依赖改为运行时 require，并默认关闭 bufferutil，减少生产冷启动兼容风险。

### 修复

- 修复“我的对比”和公开对比报告在导航高亮上的语义混用。
- 修正响应式 smoke 对人物关系页旧文案的检查，跟当前页面标题保持一致。

## 0.2.0 - 2026-06-13

### 新功能

- 新增 AI 人物情报工作台路径：动态流、话题页、机构页、全局关系图、关注列表、周报页和人物对比报告。
- 新增关注、Newsletter、ActivityEvent、CompareReport、影响力校准和质量复核相关 API、后台页面与数据模型。
- 新增生产上线门禁、迁移计划、readiness、响应式 smoke、质量复核、实体密度、关系图谱和 YouTube 字幕批处理脚本。
- 补齐目录浏览、人物详情、关系展示、近期动态、影响力拆解和代表内容的产品化入口。

### 修复

- 修复对比按钮本地状态读取造成的 hydration/lint 风险。
- 过滤公共头像出口到本地头像，减少远程图片依赖和生产图片错误。
- 清理并忽略生成类 audit/export 产物，避免仓库 footprint 继续放大。

### 文档

- 补充产品增长计划、执行板、Newsletter 配置、质量复核和上线门禁手册。
- 同步关键数据库模型说明，覆盖新增 ActivityEvent、NewsletterDeliveryLog、CompareReport 和 InfluenceScoreAuditLog。
