使用 Exa、Firecrawl、DuckDuckGo 进行网页搜索和内容抓取。

## 工具选择

| 场景 | 推荐工具 |
|-----|---------|
| 通用搜索 | `mcp__exa__web_search_exa` |
| 代码/技术文档 | `mcp__exa__get_code_context_exa` |
| 抓取特定页面 | `mcp__firecrawl__firecrawl_scrape` |
| 快速搜索验证 | `mcp__duckduckgo__search` |

## 搜索技巧

1. **人物研究**: 搜索 "{name} AI researcher" 或 "{name} machine learning"
2. **公司信息**: 搜索 "{company} AI team" 或 "{company} research"
3. **论文查找**: 搜索 "{name} paper arxiv" 或 "{name} publications"
4. **社交媒体**: 搜索 "{name} twitter" 或 "{name} linkedin"
